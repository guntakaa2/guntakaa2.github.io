Web Application Programming and Hacking (WAPH)

This course explores the development and security of modern web applications. It provides hands-on practice with technologies like HTML, CSS, JavaScript, PHP, and AJAX. Students also explore web vulnerabilities such as XSS and SQL injection, and learn to secure applications using best practices.

Hands-on Projects

  • Lab 1 – HTTP Analysis using Telnet & Wireshark
  • Lab 2 – JavaScript and DOM Manipulation
  • Hackathon 1 – XSS Attack Simulation & Defense
  • Personal Portfolio Deployment on GitHub

Skills Gained

  • Front-end development with HTML, CSS, JavaScript, jQuery
  • Client-server communication using Ajax
  • Using developer tools to debug web apps
  • Secure programming and input validation

Reflections

The WAPH course taught me not only how to build web applications, but also how to defend them against real-world threats. I particularly enjoyed the hackathons and the hands-on labs which helped me understand security issues from an attacker’s perspective.
